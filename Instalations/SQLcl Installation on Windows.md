# SQLcl Installation on Windows

## Installation Steps

### 1. Download SQLcl
- Go to https://www.oracle.com/database/sqldeveloper/technologies/sqlcl/download/
- Download `sqlcl-latest.zip`

### 2. Extract SQLcl
```
Extract sqlcl-latest.zip to C:\Oracle\sqlcl\
```

### 3. Download OpenJDK
- In Teams you will have OpenJDK.zip

### 4. Extract and Rename JDK
```
1. Extract to C:\Oracle\sqlcl\
2. Rename the folder (e.g., jdk-25.0.1) to jdk
```

### 5. Folder Structure (Result)
```
C:\Oracle\sqlcl\
├── bin\
├── lib\
├── jdk\
│   ├── bin\
│   └── lib\
└── ...
```
